#5 Ver historial de commits

  • git log = mostrar todos los guardadaos realizados.
  • git log - -oneline = Ver version Resumida con doble guion y oneline.

#6 Diferencias antes de guardar

  • git diff = para ver los cambios antes de hacer un commit.

#7 Ver repositorios remotos

  • git remote -v = verifica a que repositorio de github esta conectado tu proyecto

#8 Cambiar la URL de un repositorio

  • git remote -v = Ver la URL actual.
  • git remote set-url origin https://github.com/usuario/nuevo-repo.git = Cambiar la URL.
  • git remote -v = Comprobar la URL.
