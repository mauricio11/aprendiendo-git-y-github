#1 Navegacion basica

  • "pwd" = te indica donde estas.
  • "ls" = lista de archivos.
  • "cd" nombre_carpetas = entra en una carpeta.
  • "cd .." = volver atras.
  • "mkdir" (Nombre del proyecto) = crear carpeta dentro de VSCode.
  • "code ." = abrir VS code en esa carpeta.

#2 Empezar Git desde cero

  • "git --version" = Ver si Git esta instalado.
  • "git config --global user.name *TuNombre*" = Esta configuracion inicial se hace una vez (OBLIGATORIO).
  • "git config --global user.email *tuemail@gmail.com*" = Esta configuracion inicial se hace una vez (OBLIGATORIO).
  • "git init" = Convierte la carpeta proyecto en proyecto Git, creando tu primer repositorio.
  • "git status" = Ver el estado del proyecto, te dice que archivos han cambiado.
  • "git add ." = Añades todos los archivos a la nube.
  • "git add archivo.txt" = Añade un archivo especifico a la nube.
  • "git comit -m "primer comit" " = Guarda cambio con un mensaje agregado.

#3 GitHub para subir proyector

*Primero creas un repositorio en GitHub (desde la web)*

  • "git remote add origin https://github.com/usuario/repo.git" = Añade la URL del repositorio creado para enlazarlos.
  • "git push -u origin main" = Se utiliza para subir los cambios.
  • "git push -u origin master" = hace lo mismo del anterior pero con una rama diferente.
  • "git pull" = Trae los cambio del repositorio.

#4 Practica haciendo tu primer proyecto

    Crear carpeta.

  1. mkdir mi_primer_git

    2. Entrar en carperta.

  2. cd mi_primer_git

    3. Inicializa Git.

  3. git init

    4. Crea tu primer archivo en tu programa favorito.

  4. echo "hola mundo" > index.txt

    5. Cambia la rama en en principal para enlazarla en GitHub.

  5. git branch -m main

    6. Añadelo en el repositorio.

  6. git add .

    7. Guardalo en la nube con texto informativo

  7. git commit -m "primer commit"
